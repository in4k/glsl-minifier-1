CLI tool for optimizing GLSL.

## Installation

```sh
$ npm install -g --save glsl-optimizer
```

## Flags

### Required
	-i, --input [example: ./example/example.png] [required]
	-o, --output [example: ./example/example.dds] [required]

### Tool flags
	-f, --flags ["flag value" "flag value"] [not required]
